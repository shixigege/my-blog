约翰·伯格在《观看之道》中开篇写道："观看先于言语。"儿童先学会看，然后才能说话。但讽刺的是，成年之后我们每天都在"看"，却很少真的"观看"。

## 被设计的观看

这本书写于 1972 年，但它的核心观点在今天比任何时候都更 relevant。伯格说，我们所处的世界充斥着被设计好的图像——广告、电视、新闻照片——这些图像在潜移默化中塑造了我们的观看方式。

你以为是你在看图像，实际上是图像在看你。

伯格提出了一个令人不安的观点：在广告的世界里，所有图像都在向我们展示一种"被羡慕的生活"。你买了一辆车，不是因为你需要车，而是因为你想成为广告里那个开着车的人。

## 观看与权力

书中分析了欧洲油画中的裸体画传统。伯格指出，这些画作中女性总是以一种"被观看"的姿态出现——她的身体面向观众，她的眼神却回避着。这种观看方式暴露了权力关系：男性观看，女性被观看。

四十年后，这种分析在社交媒体的时代获得了新的含义。每天有数十亿张自拍被上传，我们既是观看者，也是被观看者。

## 重新学习观看

伯格的书让我意识到，"看"是一种需要学习的能力。

```python
def learning_to_see():
    lessons = [
        "不要只看，要观察",
        "不要只拍，要构图",
        "不要只点赞，要理解",
        "不要只消费，要质疑",
    ]
    for lesson in lessons:
        print(f"✓ {lesson}")
```

每当我觉得世界变得无聊的时候，我就会提醒自己：不是世界变无聊了，是我停止了真正的观看。路边的一棵树，每天经过时看到的样子都不一样——春天的新绿、夏天的浓荫、秋天的金黄、冬天的枯枝。

> 观看的能力，其实就是保持对世界好奇心的能力。

伯格的话到今天仍然是警醒：在这个图像爆炸的时代，最重要的不是看到更多，而是看得更深。
