白不是一种颜色，而是一种容纳性。这句话出自日本设计师原研哉的著作《白》。我一直把白当成"没有"——没有颜色，没有内容。但原研哉告诉我，白是所有可能性的总和。

## 白的哲学

一张白纸不是空的。它包含着一切尚未落笔的想法。MUJI 的设计之所以让人安静，不是因为极简，而是那些空白区域给了物品呼吸的空间。

留白不是浪费——它是给思想的栖居。

原研哉在书中写道：

> 白是一种极端的敏感。它既是颜色，也是思想的方式。

## 日常中的白

想想和纸的质感。它不像打印纸那样惨白刺眼，而是带着一种温润的、近乎透明的白。透光的时候能看到纸纤维的纹理，像是凝固的光。

日本建筑中的白也不是涂料的白，而是材料的本色——未经涂装的木头经过长时间的自然风化，呈现出一种灰白。这种白里有时间的味道。

## 数字世界的白

数字世界中最奢侈的东西是"留白"。每个像素都在争夺注意力，每个设计元素都在说"看我"。真正自信的设计敢于说不——敢于让某些区域保持空白。

```css
/* 留白的 CSS 哲学 */
.white-space {
  /* 不是浪费，是投资 */
  padding: 4rem 2rem;
  margin-bottom: 3rem;
  /* 给眼睛一个休息的地方 */
}
```

苹果的网页设计就是留白的典范。它敢于在产品图片周围留出大量空白，让产品本身成为焦点。这不是因为懒惰，而是因为克制。

### 为什么要学会留白

写文章也是一样。好的文章不在于说了多少，而在于没说什么。懂得在合适的地方停下来，给读者留出思考的空间。这个空间，就是文字的留白。

> 留白不是空无一物，而是充满可能性的沉默。
