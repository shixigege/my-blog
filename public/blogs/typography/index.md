谷登堡未必能料到，他在十五世纪摆弄的那些铅字，会在五百年后以一种他完全无法理解的方式影响着数十亿人的阅读。字距、行高、栏宽——这些参数在铅字时代是物理约束，在数字时代却成了设计选择。当一切皆可变，我们反而失去了那种"被限制的美"。

## 活字的哲学

宋人刻书的匠人，一笔一画刻入梨木板，试印，修正，再试印——每一版都是一次无法撤销的决定。这种对字的敬畏，是数字排版最需要找回的品质。

活字不是工具，是手艺。

```python
# 一个简单的比喻：排版如同排版代码
def typeset(text, columns=2, leading=1.5):
    """排版引擎的核心逻辑"""
    words = text.split()
    column_height = len(words) // columns
    for col in range(columns):
        col_words = words[col*column_height:(col+1)*column_height]
        print(" ".join(col_words))
        if col < columns - 1:
            print("—" * 20)
```

## 铅字时代的限制之美

在金属活字的时代，字体大小是物理的——你不能随意拉大一个标题，你需要更大尺寸的铅字。行距由铅条决定，栏宽由版心决定。这些限制反而催生了印刷术的黄金时代。

而今天，一个高中生打开 Google Docs 就能做出五百年前只有顶尖工匠才能完成的排版。但问题是——做出来的东西，往往没有那么好。

### 可读性的秘密

优秀的排版是不可见的。当你读一本排版精良的书时，你不会注意到字距、行高、栏宽——你只会沉浸在文字里。排版最好的状态，就是让读者忘记排版的存在。

> 排版的艺术，在于让读者只看到内容，看不到形式。

## 数字时代的排版挑战

响应式设计让排版变得更加复杂。同一篇文章，在手机上是一栏，在平板上是两栏，在桌面上可能是三栏。这种灵活性的代价是，我们再也无法像铅字时代那样精确控制每一行的长度。

但这不意味着我们应该放弃对排版的追求。相反，正因为一切皆可变，我们更应该理解什么不可变——可读性、节奏感、呼吸感。
