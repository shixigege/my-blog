要加的交互（实际实现）

暗色切换 右上角"暗色/亮色"按钮，CSS class 切换，存 localStorage 记住用户选择
hover 下划线 卡片底部一根朱砂红线，默认隐藏 scaleX(0)，hover 时展开 scaleX(1)
hover 背景 卡片 hover 时 background 微亮，有"触感"反馈
导航高亮 当前页加朱砂色下划线，一眼知道自己在哪
暗色配色 底 #1E1C18 / 卡片 #282420 / 文字 #C8C0B0 / 标题 #D0C8B8 / 保持朱砂红
